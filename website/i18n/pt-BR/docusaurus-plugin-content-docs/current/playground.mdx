# Playground

O Playground permite que você rode o Docusaurus **em seu navegador, sem precisar instalar nada**!

Eles são úteis principalmente para:

- Testando o Docusaurus
- Relatando bugs

Use [docusaurus.new](https://docusaurus.new) como um atalho fácil de lembrar.

Escolha uma das opções disponíveis abaixo.

```mdx-code-block
import {PlaygroundCardsRow} from '@site/src/components/Playground';

<PlaygroundCardsRow />
```

:::tip

Por conveniência, lembraremos de sua escolha na próxima vez que visitar [docusaurus.new](https://docusaurus.new).

:::
