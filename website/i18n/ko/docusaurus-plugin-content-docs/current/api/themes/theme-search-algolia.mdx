# 📦 theme-search-algolia

알골리아 문서 검색 통합을 위한 `@theme/SearchBar` 컴포넌트를 지원하는 테마입니다. `@docusaurus/theme-classic`에서 좀 더 쉽게 검색 기능을 통합할 수 있습니다. 자세한 내용은 [검색](../../search.mdx) 관련 문서를 참고하세요.

```bash npm2yarn
npm install --save @docusaurus/theme-search-algolia
```

OpenSearch 기능 지원과 함께 `/search`(스위즐할 수 있는 `SearchPage` 컴포넌트)에서 사용할 수 있는 검색 페이지를 추가할 수 있습니다. `themeConfig.algolia.searchPagePath`를 통해 기본 경로를 변경할 수 있습니다. 검색 페이지를 비활성화하려면 `false` 값을 설정합니다.

:::tip

`@docusaurus/preset-classic`을 설치한 경우에는 테마를 따로 설치할 필요는 없습니다.

:::
