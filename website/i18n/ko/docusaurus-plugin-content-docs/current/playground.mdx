# 플레이그라운드

플레이그라운드는 **별도의 설치 없이 여러분의 브라우저에서** 도큐사우루스를 직접 실행해볼 수 있습니다!

이런 경우에 유용하게 사용할 수 있습니다.

- 도큐사우루스 테스트하기
- 버그 등록

기억하기 쉬운 단축 주소로 [docusaurus.new](https://docusaurus.new)를 사용하세요.

아래에서 사용하기 원하는 옵션을 선택하세요.

```mdx-code-block
import {PlaygroundCardsRow} from '@site/src/components/Playground';

<PlaygroundCardsRow />
```

:::tip

여러분의 편의를 위해 다음에 [docusaurus.new](https://docusaurus.new)를 방문할 때는 선택한 옵션을 자동으로 적용할 겁니다.

:::
