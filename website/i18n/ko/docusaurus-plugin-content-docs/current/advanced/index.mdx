# 고급 튜토리얼

이번 섹션은 세심하게 구성되지는 않았지만 다음과 같은 주제를 다룹니다.

```mdx-code-block
import DocCardList from '@theme/DocCardList';

<DocCardList />
```

기본 가이드를 숙지하고 플러그인 구성 방법, 리액트 컴포넌트 작성 방법 등과 같은 기본 사항은 알고 있다고 가정합니다. 이번 섹션은 플러그인 작성자와 코드 기여자가 참여하며 간혹 [플러그인 API](../api/plugin-methods/README.mdx)나 상세한 아키텍처를 참조할 수 있습니다. 모든 것을 이해하지 못하더라도 좌절하지 마세요😉
