# Ressources formidables

Une liste de projets communautaires Docusaurus intéressants.

## Documentation communautaire {#community-documentation}

Le site **[Docusaurus.community](https://docusaurus.community)** complète la [documentation officielle](/docs) avec des exemples plus complexes et des articles complets qui recommandent les meilleures pratiques et fournissent des solutions aux problèmes courants.

Il s'agit également d'une application Docusaurus complète, tout comme notre site principal, de sorte que les exemples de personnalisation et d'expérimentation sont parfaitement adaptés pour être hébergés à cet endroit.

:::info

Nous pourrions autoriser l'ajout de liens directs dans notre documentation officielle qui renvoient à la documentation de la communauté, au cas par cas, si la communauté trouve ces liens utiles.

Nous envisageons également de l'intégrer dans la documentation officielle à l'avenir.

:::

## Plugins de la communauté {#community-plugins}

### Recherche {#search}

- [docusaurus-plugin-lunr](https://github.com/daldridge/docusaurus-plugin-lunr) - extension Docusaurus pour créer un index de recherche local à utiliser avec Lunr.js
- [docusaurus-lunr-search](https://github.com/lelouch77/docusaurus-lunr-search) - Recherche hors ligne pour Docusaurus
- [docusaurus-lunr-search](https://github.com/cmfcmf/docusaurus-search-local) - Recherche locale/hors ligne pour Docusaurus
- [@easyops-cn/docusaurus-search-local](https://github.com/easyops-cn/docusaurus-search-local) - Recherche hors ligne/locale de Docusaurus (langage de zh supporté)
- [docusaurus-theme-search-typesense](https://github.com/typesense/docusaurus-theme-search-typesense) - Plugin Docusaurus pour [Typesense DocSearch](https://typesense.org/docs/latest/guide/docsearch.html)
- [@markprompt/docusaurus-theme-search](https://github.com/motifland/markprompt-js/tree/main/packages/docusaurus-theme-search) - Plugin Docusaurus pour ajouter une recherche documentaire générative alimentée par l'IA / LLM, propulsée par [Markprompt](https://markprompt.com).
- [meilisearch-docsearch](https://github.com/tauri-apps/meilisearch-docsearch) - Plugin Docusaurus pour [Meilisearch](https://www.meilisearch.com)
- [@orama/plugin-docusaurus](https://github.com/askorama/orama/tree/main/packages/plugin-docusaurus) - Plugin [Orama](https://askorama.ai/) pour Docusaurus v2
- [@orama/plugin-docusaurus-v3](https://github.com/askorama/orama/tree/main/packages/plugin-docusaurus-v3) - Plugin [Orama](https://askorama.ai/) pour Docusaurus v3
- [@getcanary/docusaurus-theme-search-pagefind](https://getcanary.dev/docs/integrations/docusaurus.html) - Créer un index [Pagefind](https://pagefind.app/) et utilise [Canary](https://github.com/fastrepl/canary) comme primitives d'interface utilisateur.

### Intégrations {#integrations}

- [docusaurus2-dotenv](https://github.com/jonnynabors/docusaurus2-dotenv) - Une extension Docusaurus supportant dotenv et autres variables d'environnement
- [posthog-docusaurus](https://github.com/PostHog/posthog-docusaurus) - Intégrer le produit analytique [PostHog](https://posthog.com/) avec Docusaurus
- [docusaurus-plugin-moesif](https://github.com/Moesif/docusaurus-plugin-moesif) - Ajoute [Moesif API Analytics](https://www.moesif.com/) pour observer le comportement des utilisateurs et identifier où les développeurs se rendent dans votre entonnoir d'activation.
- [docusaurus-plugin-yandex-metrica](https://github.com/sgromkov/docusaurus-plugin-yandex-metrica) - Ajoute le compteur [Yandex Metrika](https://metrika.yandex.ru/) pour évaluer le trafic du site et analyser le comportement des utilisateurs.

### Fonctionnalités {#features}

- [docusaurus-theme-github-codeblock](https://github.com/saucelabs/docusaurus-theme-github-codeblock). Une extension dans Docusaurus permettant de référencer des exemples de code provenant de répertoires publics GitHub
- [docs-to-pdf](https://github.com/jean-humann/docs-to-pdf) - Générer de la documentation au format PDF
- [docusaurus-plugin-sass](https://github.com/rlamana/docusaurus-plugin-sass) - Soutient les feuilles de style Sass/SCSS
- [docusaurus-plugin-remote-content](https://github.com/rdilweb/docusaurus-plugin-remote-content) - Une extension Docusaurus qui vous permet de récupérer du contenu à partir de sources distantes
- [@graphql-markdown/docusaurus](https://github.com/graphql-markdown/graphql-markdown) - Un plugin Docusaurus pour générer de la documentation à partir de schémas GraphQL
- [docusaurus-plugin-react-docgen-typescript](https://github.com/atomicpages/docusaurus-plugin-react-docgen-typescript) - Un plugin Docusaurus pour charger des annotations de type depuis `react-docgen-typescript`
- [docusaurus-plugin-react-docgen](https://github.com/atomicpages/docusaurus-plugin-react-docgen) - Un plugin Docusaurus pour charger des annotations de type depuis `react-docgen`
- [docusaurus-plugin-module-alias](https://github.com/atomicpages/docusaurus-plugin-module-alias) - Un plugin Docusaurus pour faire rapidement des alias des modules locaux
- [docusaurus-protobuffet](https://github.com/protobuffet/docusaurus-protobuffet) - Ensemble d'outils Docusaurus pour la documentation des contrats Protobuf
- [docusaurus-prince-pdf](https://github.com/signcl/docusaurus-prince-pdf) - Génère des PDF avec PrinceXML pour un meilleur sous-ensemble de polices et des fonctionnalités de Table des matières. Prise en charge des sites Docusaurus
- [redocusaurus](https://github.com/rohit-gohri/redocusaurus) - Un preset Docusaurus pour intégrer la documentation OpenAPI dans vos docs avec [Redoc](https://github.com/redocly/redoc)
- [plugin-image-zoom](https://github.com/flexanalytics/plugin-image-zoom) - Un plugin de zoom d'image pour Docusaurus
- [docusaurus-plugin-typedoc](https://github.com/tgreyuk/typedoc-plugin-markdown/tree/master/packages/docusaurus-plugin-typedoc) - Un plugin Docusaurus pour construire de la documentation avec [TypeDoc](https://typedoc.org/)
- [docusaurus-openapi-docs](https://github.com/PaloAltoNetworks/docusaurus-openapi-docs) - Un plugin Docusaurus et un thème pour la génération de docs OpenAPI interactifs
- [docusaurus-post-generator](https://github.com/moojing/docusaurus-post-generator) - Un outil en ligne de commande pour l'utilisateur afin d'ajouter un fichier blog/doc rapidement par une commande comme `yarn gen-post new [template] [post_name]`.
- [docusaurus-graph](https://github.com/Arsero/docusaurus-graph) - Un plugin Docusaurus pour générer un graphique montrant les liens entre les fichiers de documentation.
- [docusaurus-i18n](https://github.com/moonrailgun/docusaurus-i18n) - Auto translate docusaurus documents with openai.
