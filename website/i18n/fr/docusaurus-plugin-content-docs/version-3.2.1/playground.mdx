# Bac à sable

Les bacs à sable vous permettent d'exécuter Docusaurus **dans votre navigateur, sans rien installer **!

Ils sont surtout utiles pour :

- Tester Docusaurus
- Signaler des bugs

Utilisez [docusaurus.new](https://docusaurus.new) comme raccourci facile à mémoriser.

Choisissez l'une des options disponibles ci-dessous.

```mdx-code-block
import {PlaygroundCardsRow} from '@site/src/components/Playground';

<PlaygroundCardsRow />
```

:::tip

Pour plus de commodité, nous nous souviendrons de votre choix la prochaine fois que vous visiterez [docusaurus.new](https://docusaurus.new).

:::
