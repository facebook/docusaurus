# Montée de version de Docusaurus

Le versionnage de Docusaurus est basé sur le schéma `major.minor.patch` et respecte le [**Versionnage sémantique**](https://semver.org/).

Les **changements de rupture** ne sont publiés qu'à l'occasion de mises à jour de versions majeures, et sont documentés de manière détaillée dans les guides de mise à jour suivants.
