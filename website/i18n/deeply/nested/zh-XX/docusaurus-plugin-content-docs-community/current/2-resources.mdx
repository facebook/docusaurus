# 社区精选

下方是 Docusaurus 社区项目精选列表。

## 社区文档 {#community-documentation}

**[Docusaurus.community](https://docusaurus.community)** 站点通过更复杂的示例和完整的文章扩展了[官方文档](/docs)，推荐最佳实践并提供常见问题的解决方案。

它也是一个完整的 Docusaurus 应用程序，就像我们的主站点一样，因此定制和实验示例完全可以托管在这里。

:::info

如果社区认为此类文章链接有用，我们可能会视具体情况允许在官方文档中添加指向社区文档的直接链接。

我们还考虑在未来的某个时候将其进一步整合到官方文档中。

:::

## 社区插件 {#community-plugins}

### 搜索 {#search}

- [docusaurus-plugin-lunr](https://github.com/daldridge/docusaurus-plugin-lunr) - 创建本地搜索索引并配合 Lunr.js 使用的 Docusaurus 插件
- [docusaurus-lunr-search](https://github.com/lelouch77/docusaurus-lunr-search) - Docusaurus 离线搜索
- [docusaurus-search-local](https://github.com/cmfcmf/docusaurus-search-local) - Docusaurus 离线/本地搜索
- [@easyops-cn/docusaurus-search-local](https://github.com/easyops-cn/docusaurus-search-local) - Docusaurus 离线/本地搜索（支持中文）
- [docusaurus-theme-search-typesense](https://github.com/typesense/docusaurus-theme-search-typesense) - [Typesense DocSearch](https://typesense.org/docs/latest/guide/docsearch.html)的Docusaurus插件
- [@markprompt/docusaurus-theme-search](https://github.com/motifland/markprompt-js/tree/main/packages/docusaurus-theme-search) - 为文档添加生成式 AI / 大语言模型搜索的 Docusaurus 插件，由 [Markprompt](https://markprompt.com) 支持。
- [meilisearch-docsearch](https://github.com/tauri-apps/meilisearch-docsearch) - [Meilisearch](https://www.meilisearch.com)的 Docusaurus 插件
- [@orama/plugin-docusaurus](https://github.com/askorama/orama/tree/main/packages/plugin-docusaurus) - 适配 Docusaurus v2 的[Orama](https://askorama.ai/) 插件
- [@orama/plugin-docusaurus-v3](https://github.com/askorama/orama/tree/main/packages/plugin-docusaurus-v3) - 适配Docusaurus v3的 [Orama](https://askorama.ai/) 插件
- [@getcanary/docusaurus-theme-search-pagefind](https://getcanary.dev/docs/integrations/docusaurus.html) - 创建 [Pagefind](https://pagefind.app/) 索引并使用 [Canary](https://github.com/fastrepl/canary) 作为原始 UI 。

### 集成 {#integrations}

- [docusaurus2-dotenv](https://github.com/jonnynabors/docusaurus2-dotenv) - 支持 dotenv 及其他环境变量的 Docusaurus 插件
- [posthog-docusaurus](https://github.com/PostHog/posthog-docusaurus) - 将[PostHog](https://posthog.com/) 产品分析功能集成到 Docusaurus
- [docusaurus-plugin-moesif](https://github.com/Moesif/docusaurus-plugin-moesif) - 添加 [Moesif API 分析](https://www.moesif.com/)来追踪用户行为并定位开发者所浏览的页面。
- [docusaurus-plugin-yandex-metrica](https://github.com/sgromkov/docusaurus-plugin-yandex-metrica) - 添加 [Yandex Metrika](https://metrika.yandex.ru/) 来评估网站流量并分析用户行为。

### 特性 {#features}

- [docusaurus-theme-github-codeblock](https://github.com/saucelabs/docusaurus-theme-github-codeblock) -  支持从 GitHub 公共仓库引用代码的 Docusaurus 插件
- [docs-to-pdf](https://github.com/jean-humann/docs-to-pdf) - 生成 PDF 格式的文档
- [docusaurus-plugin-sass](https://github.com/rlamana/docusaurus-plugin-sass) - Sass/SCSS 样式表支持
- [docusaurus-plugin-remounte-content](https://github.com/rdilweb/docusaurus-plugin-remote-content) - 允许您从远程来源获取内容的 Docusaurus 插件
- [@graphql-markdown/docusaurus](https://github.com/graphql-markdown/graphql-markdown) - 从 GraphQL schemas 生成文档的插件
- [docusaurus-plugin-react-docgen-typescript](https://github.com/atomicpages/docusaurus-plugin-react-docgen-typescript) - 从 `react-docgen-typescript` 中加载类型提示的 Docusaurus插件
- [docusaurus-plugin-react-docgen](https://github.com/atomicpages/docusaurus-plugin-react-docgen) - 一个从 `react-docgen`加载类型提示的插件
- [docusaurus-plugin-module-alias](https://github.com/atomicpages/docusaurus-plugin-module-alias) - 快速制定本地模块别名的插件
- [docusaurus-protobuffet](https://github.com/protobuffet/docusaurus-protobuffet) - 在 Docusaurus 中显示 Protobuf 协议文档的插件
- [docusaurus-princ-pdf](https://github.com/signcl/docusaurus-prince-pdf) - 使用 PrinceXML 生成 PDF 以获取更好的字体子集和 ToC 功能。 支持 Docusaurus 站点
- [redocusaurus](https://github.com/rohit-gohri/redocusaurus) - 用 [Redoc](https://github.com/redocly/redoc) 将 OpenAPI 文档集成到你的文档的 Docusaurus 预设
- [plugin-image-zoom](https://github.com/flexanalytics/plugin-image-zoom) - 一个适用于 Docusaurus 的图像缩放插件
- [docusaurus-plugin-typedoc](https://github.com/tgreyuk/typedoc-plugin-markdown/tree/master/packages/docusaurus-plugin-typedoc) - 使用 [TypeDoc](https://typedoc.org/) 构建文档的Docusaurus 插件
- [docusaurus-openapi-docs](https://github.com/PaloAltoNetworks/docusaurus-openapi-docs) - 生成交互式 OpenAPI 文档的 Docusaurus 插件和主题
- [docusaurus-post-generator](https://github.com/moojing/docusaurus-post-generator) - 一个命令行工具，用户可以通过类似 `yarn gen-pose new [template] [post_name]` 的指令来快速添加 blog/doc 文件
- [docusaurus-graph](https://github.com/Arsero/docusaurus-graph) - 一个用于显示文档关系图的插件
- [docusaurus-i18n](https://github.com/moonrailgun/docusaurus-i18n) - 自动使用 OpenAI 翻译docusaurus 文档
