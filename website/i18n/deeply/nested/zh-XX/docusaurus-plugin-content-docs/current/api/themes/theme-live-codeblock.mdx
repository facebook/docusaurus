# 📦 theme-live-codeblock

这个主题提供了由 [react-live](https://commerce.nearform.com/open-source/react-live/) 驱动的 `@theme/CodeBlock` 组件。 你可以在[交互式代码编辑器](../../guides/markdown-features/markdown-features-code-blocks.mdx#interactive-code-editor)的文档中了解详情。

```bash npm2yarn
npm install --save @docusaurus/theme-live-codeblock
```

### 配置 {#configuration}

```js title="docusaurus.config.js"
export default {
  plugins: ['@docusaurus/theme-live-codeblock'],
  themeConfig: {
    liveCodeBlock: {
      /**
       * 实时效果显示的位置，在编辑器上方还是下方
       * 可选参数："top" | "bottom"
       */
      playgroundPosition: 'bottom',
    },
  },
};
```
