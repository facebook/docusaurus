# 代码沙盒

沙盒允许你**不需要安装任何东西**，就可以直接在浏览器中试用 Docusaurus！

沙盒的主要用途：

- 测试 Docusaurus
- 漏洞反馈

您可以前往简单好记的 [docusaurus.new](https://docusaurus.new) 来快速使用此功能。

您可以选择下列任意一个沙盒：

```mdx-code-block
import {PlaygroundCardsRow} from '@site/src/components/Playground';

<PlaygroundCardsRow />
```

:::tip

当您下次访问上述链接时，我们会记住您的选择。

:::
