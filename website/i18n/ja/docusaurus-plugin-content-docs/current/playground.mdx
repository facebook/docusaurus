# プレイグラウンド

プレイグラウンドを利用すれば、**お使いのブラウザ上で、何のインストールも必要とせずに**Docusaurusの実行ができます！

主に次の用途で便利です。

- Docusaurusのテスト
- バグの報告

覚えやすい短縮URLとして、[docusaurus.new](https://docusaurus.new)を用意していますので、ご利用ください。

次の利用可能な選択肢から、1つをお選びください。

```mdx-code-block
import {PlaygroundCardsRow} from '@site/src/components/Playground';

<PlaygroundCardsRow />
```

:::tip

利便性向上のため、[docusaurus.new](https://docusaurus.new)に次回アクセスした時には、今回選択したものを開きます。

:::
