# 優れたリソースへのリンク集

Docusaurus コミュニティプロジェクトの選りすぐりのリストです。

## コミュニティによるドキュメント {#community-documentation}

サイト**[Docusaurus.community](https://docusaurus.community)** は、より複雑な例や、ベストプラクティスを推奨しよく起こる問題の解決方法を提供する完全な記事により[公式ドキュメント](/docs)を拡張します。

また、これは私たちのメインサイトと同じように完全なDocusaurusアプリケーションなので、カスタマイズ及び実験例はそこにホストされるのが最適です。

:::info

コミュニティがそのような記事へのリンクが有用であると判断した場合、ケースバイケースですが、公式文書にコミュニティ文書を指す直接リンクを追加することを許可するかもしれません。

また、将来のある時点で公式文書にさらに統合することも検討しています。

:::

## コミュニティによって開発されたプラグイン {#community-plugins}

### 検索 {#search}

- [docusaurus-plugin-lunr](https://github.com/daldridge/docusaurus-plugin-lunr) - ローカル検索インデックスを作成するための Lunr.js 向けのDocusaurusプラグイン
- [docusaurus-lunr-search](https://github.com/lelouch77/docusaurus-lunr-search) - Docusaurus用オフライン検索
- [docusaurus-search-local](https://github.com/cmfcmf/docusaurus-search-local) - Docusaurus用オフライン/ローカル検索
- [@easyops-cn/docusaurus-search-local](https://github.com/easyops-cn/docusaurus-search-local) - Docusaurus用のオフライン / ローカル検索プラグイン（中国語対応）
- [docusaurus-theme-search-typesense](https://github.com/typesense/docusaurus-theme-search-typesense) - [Typesense DocSearch](https://typesense.org/docs/latest/guide/docsearch.html)のためのDocusaurusプラグイン
- [@markprompt/docusaurus-theme-search](https://github.com/motifland/markprompt-js/tree/main/packages/docusaurus-theme-search) - [Markprompt](https://markprompt.com)Markpromptを利用したAI/LLMによるドキュメント検索機能を追加するDocusaurusプラグイン。
- [meilisearch-docsearch](https://github.com/tauri-apps/meilisearch-docsearch) - [Meilisearch](https://www.meilisearch.com)のためのDocusaurusプラグイン
- [@orama/plugin-docusaurus](https://github.com/askorama/orama/tree/main/packages/plugin-docusaurus) - Docusaurus v2用の[Orama](https://askorama.ai/)プラグイン
- [@orama/plugin-docusaurus-v3](https://github.com/askorama/orama/tree/main/packages/plugin-docusaurus-v3) - Docusaurus v3用の[Orama](https://askorama.ai/)プラグイン
- [@getcanary/docusaurus-theme-search-pagefind](https://getcanary.dev/docs/integrations/docusaurus.html) - [Pagefind](https://pagefind.app/) でインデックスを生成し、[Canary](https://github.com/fastrepl/canary) を UI の基本コンポーネントとして利用します。

### インテグレーション {#integrations}

- [docusaurus2-dotenv](https://github.com/jonnynabors/docusaurus2-dotenv) - dotenv やその他の環境変数をサポートするDocusaurusプラグイン
- [poshog-docusaurus](https://github.com/PostHog/posthog-docusaurus) - [PostHog](https://posthog.com/) 製品分析と Docusaurus の統合
- [docusaurus-plugin-moesif](https://github.com/Moesif/docusaurus-plugin-moesif) - [Moesif API Analytics](https://www.moesif.com/) を追加してユーザーの行動を追跡し、開発者がアクティベーションファンネルのどこにドロップするかを特定できるようにするプラグイン
- [docusaurus-plugin-yandex-metrica](https://github.com/sgromkov/docusaurus-plugin-yandex-metrica) - サイトのトラフィックを評価し、ユーザの行動を分析する [Yandex Metrika](https://metrika.yandex.ru/) プラグイン

### 特徴 {#features}

- [docusaurus-theme-github-codeblock](https://github.com/saucelabs/docusaurus-theme-github-codeblock). GitHub のパブリックリポジトリからコードの例を参照するDocusaurus v2プラグイン
- [docs-to-pdf](https://github.com/jean-humann/docs-to-pdf) - ドキュメントをPDF形式で生成
- [docusaurus-plugin-sass](https://github.com/rlamana/docusaurus-plugin-sass) - Sass/SCSS スタイルシートをサポートするプラグイン
- [docusaurus-plugin-remote-content](https://github.com/rdilweb/docusaurus-plugin-remote-content) - 外部のソースからコンテンツを取得できるようにするDocusaurusプラグイン
- [@graphql-markdown/docusaurus](https://github.com/graphql-markdown/graphql-markdown) - GraphQLスキーマからドキュメントを生成するためのDocusaurusプラグイン
- [docusaurus-plugin-react-docgen-typescript](https://github.com/atomicpages/docusaurus-plugin-react-docgen-typescript) - `react-docgen-typescript` から型注釈を読み込むためのDocusaurusプラグイン
- [docusaurus-plugin-react-docgen](https://github.com/atomicpages/docusaurus-plugin-react-docgen) - `react-docgen` から型注釈を読み込むための Docusaurus プラグイン
- [docusaurus-plugin-module-alias](https://github.com/atomicpages/docusaurus-plugin-module-alias) - ローカルモジュールのエイリアスを簡単に設定するための Docusaurus プラグイン
- [docusaurus-protobufficient](https://github.com/protobuffet/docusaurus-protobuffet) - Protobuf contract のドキュメント向けの Docusaurus ツールセット
- [docusaurus-prince-pdf](https://github.com/signcl/docusaurus-prince-pdf) - フォントのサブセット化及び目次機能の向上のためにPrinceXMLでPDFを生成。 （Docusaurus製サイト対応）
- [redocusaurus](https://github.com/rohit-gohri/redocusaurus) - OpenAPI ドキュメントを [Redoc](https://github.com/redocly/redoc) でドキュメントに統合するための Docusaurus プリセット
- [plugin-image-zoom](https://github.com/flexanalytics/plugin-image-zoom) - Docusaurus用画像拡大プラグイン
- [docusaurus-plugin-typedoc](https://github.com/tgreyuk/typedoc-plugin-markdown/tree/master/packages/docusaurus-plugin-typedoc) - ドキュメントを[TypeDoc](https://typedoc.org/)で構築するためのDocusaurusプラグイン
- [docusaurus-openapi-docs](https://github.com/PaloAltoNetworks/docusaurus-openapi-docs) - 対話的なOpenAPIドキュメントを生成するためのDocusaurusプラグイン及びテーマ
- [docusaurus-post-generator](https://github.com/moojing/docusaurus-post-generator) - ユーザが`yarn gen-post new [template] [post_name]`のようなコマンドでblog/docファイルを手早く追加できるようにするコマンドラインツール
- [docusaurus-graph](https://github.com/Arsero/docusaurus-graph) - ドキュメントファイル間のリンクをグラフ表示するための Docusaurus プラグイン。
- [docusaurus-i18n](https://github.com/moonrailgun/docusaurus-i18n) - OpenAI で Docusaurus のドキュメントを自動翻訳します。
