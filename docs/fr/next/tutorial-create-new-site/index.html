Créer un nouveau site

Dans cette section, nous allons faire fonctionner notre site Docusaurus pour le développement local. Le processus ne prend que quelques minutes.

{.docImage}

Conception du site

  1. cd dans le répertoire de votre dépôt local.
cd docusaurus-tutorial
  1. Exécutez la commande docusaurus-init dans votre terminal.
docusaurus-init

L'étape Lier les dépendances... (Linking dependencies...) peut prendre un certain temps, mais elle finira par se terminer.

Le contenu suivant sera créé dans votre répertoire courant. Quelques exemples de pages de documentation (sous docs) et des articles de blog (sous website/blog) sont inclus.

├── Dockerfile
├── docker-compose.yml
├── docs
│   ├── doc1.md
│   ├── doc2.md
│   ├── doc3.md
│   ├── exampledoc4.md
│   └── exampledoc5.md
└── website
    ├── README.md
    ├── blog
    │   ├── 2016-03-11-blog-post.md
    │   ├── 2017-04-10-blog-post-two.md
    │   ├── 2017-09-25-testing-rss.md
    │   ├── 2017-09-26-adding-rss.md
    │   └── 2017-10-24-new-version-1.0.0.md
    ├── core
    │   └── Footer.js
    ├── package.json
    ├── pages
    │   └── en
    │       ├── help.js
    │       ├── index.js
    │       └── users.js
    ├── sidebars.json
    ├── siteConfig.js
    ├── static
    │   ├── css
    │   │   └── custom.css
    │   └── img
    │       ├── docusaurus.svg
    │       ├── favicon
    │       │   └── favicon.ico
    │       ├── favicon.png
    │       └── oss_logo.png
    └── yarn.lock
  1. Exécutez cd website pour aller dans le répertoire website.
  2. Exécutez npm start ou yarn start.

Une fenêtre de navigateur s'ouvrira à l'adresse http://localhost:3000.

Félicitations, vous venez de faire votre premier site Docusaurus ! Cliquez sur les pages pour vous faire une idée.
